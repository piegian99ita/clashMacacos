This REPO is designed to update the activity information of the Macacos Clash of Clans members; it's performed by using github actions and the code was written by using coc.py APIs. 
If you need to see the data there are 2 separate xlsx files that you can download locally: reward.xlsx and cc_cg_events.xlsx

REWARD.XLSX:
This file includes the data of the CWL previous wars and the data of the random classical wars.

CC_CG_EVENTS.XLSX:
This file contains the activity informations about Clan Games, Clan Capital assaults and clan events that occurs occasionally.
